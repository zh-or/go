		2. 删除

	4. 游记照片管理(下载)


** 备注: 用户昵称重复判断方法, 在user表里添加一个存储 hash(username) 的字段, 判断时只需要查询 hash(username) 从得到接结果里面判断 字符串是否相等即可, hash 是数字类型的, 索引后查询速度会很快.
